Über

Diffuse ist ein graphisches Werkzeug zum Zusammenführen und Vergleichen von Textdateien. Diffuse kann eine beliebige Anzahl Dateien direkt miteinander vergleichen und gibt dem Benutzer die Möglichkeit manuell die Zeilen-Übereinstimmung und direkt die Dateien zu bearbeiten. Diffuse kann auch Revisionen von Dateien von Bazaar, CVS, Darcs, Git, Mercurial, Monotone, RCS, Subversion, und SVK Repositorien zum Vergleichen und Zusammenführen empfangen.

Features

  • Die Möglichkeit eine beliebige Anzahl Dateien direkt miteinander vergleichen (N-Wege Zusammenführen)
  • Zeilen-Übereinstimmung kann manuell vom Benutzer korregiert werden
  • Die Möglichkeit Dateien direkt zu bearbeiten
  • Syntax Hervorhebung
  • Bazaar, CVS, Darcs, Git, Mercurial, Monotone, RCS, Subversion, und SVK Unterstützung
  • Unterstützung für UTF-8 kodierten Unicode
  • Unbegrenztes rückganig machen
  • einfache Tastatur Navigation

Anforderungen

Diffuse sollte auf jedem POSIX Betriebssystem laufen, inklusive Mac OS X, wenn die folgenden Pakete installiert sind:

  • Python >= 2.4
  • PyGTK >= 2.10

Vorkompilierte Pakete für Microsoft Windows enthalten alle Abhängigkeiten.
