0.3.0

Veröffentlicht am 2009-03-03

Die Regeln für das Parsen von Dateien in ~/.diffuse hat sich in der 0.3.0 Veröffentlichung geändert. Nicht-schwerwiegende Fehler können gemeldet werden beim Parsen alter Dateien. Diese Fehler können behoben werden durch das Entfernen fehlerhaften Zeilen (oder der ganzen Datei) aus ~/.diffuse/diffuserc. Fehler in ~/.diffuse/config werden automatisch behoben, wenn Diffuse beendet wird.

  • Neues Windows Installationsprogramm
  • Benachrichtigung beim Fokussieren wenn sich Dateien auf Festplatte ändern
  • Menüpunkte zum Einstellen der Markierung

  • '-c' Option funktioniert nun mit CVS Revisions Nummern
  • Fenstertitel beschreibt nun den aktuellen Reiter
  • Suchen-Einstellungen bleiben nun über mehrere Sitzungen bestehen
  • Kleinere Fehlerkorrekturen
