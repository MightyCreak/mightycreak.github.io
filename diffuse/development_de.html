Mailing Listen

Sie können die Entwickler über die diffuse-users mailing list erreichen für jede Benutzungs- oder Entwicklungs- Diskussion. Archive von vergangenen Diskussionen sind ebenfalls verfügbar.

Issue Tracker

Sie können den Issue Tracker auf SourceForge.net benutzen um Fehlermeldungen, Feature-Anfragen und Patches zu senden.

Entwickler Version

Die Entwickler Version von Diffuse ist hauptsächlich für Entwickler gemacht und kann instabil sein. Der Entwickler Quelltext befindet sich in einem Subversion Repositorium. Benutzen Sie diesen Befehl um eine lokale Kopie des Quelltextes zu bekommen:

$ svn co https://diffuse.svn.sourceforge.net/svnroot/diffuse/trunk diffuse

Das Subversion Repositorium auch über die Web-Oberfläche erreicht werden.

Mitarbeiten

Wenn Sie sich für die Programmierung interessieren, können Sie einen Blick auf die vorhandenen Fehlermeldungen und Feature Anfragen werfen um Ideen zu bekommen oder Programmieren Sie einfach etwas und stellen Sie es zur Diskussion auf der Mailing Liste oder dem Bug Tracker. Wenn Sie sich für die Übersetzung der Benutzeroberfläche interessieren, lesen Sie bitte die "README" Datei im "translations" Verzeichnis.

Werkzeuge bei der Diffuse Entwicklung

  • Interpreter: Python
  • Benutzeroberfläche: PyGTK
  • Dokumentationsverarbeitung: xsltproc und DocBook stylesheets
  • Microsoft Windows ausführbare Datei: py2exe
  • Microsoft Windows Installationsprogramm: Inno Setup und ADD_PATH
