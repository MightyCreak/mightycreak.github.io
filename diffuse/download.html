Download

GNU/Linux
deb rpm

The most recent release of Diffuse is provided as .deb and .rpm packages. These packages can be installed using the following commands:

  • Debian / Ubuntu: dpkg -i diffuse*.deb
  • Fedora / OpenSUSE: rpm -i diffuse*.rpm

Alternatively, Diffuse can be installed by the package management system of most major Linux distributions using the following commands:

  • Arch Linux: pacman -S diffuse
  • Debian / Ubuntu: apt-get install diffuse
  • Fedora: yum install diffuse
  • Gentoo: emerge diffuse
  • OpenSUSE: zypper install diffuse
Microsoft Windows
Win32

The Win32 installer for Diffuse contains all dependencies required for use on 32-bit and 64-bit versions of XP / Vista / Windows 7.

Mac OS X
MacPorts Fink

Packages for Mac OS X are maintained by third parties. Diffuse can be installed using the following commands:

  • MacPorts: port install diffuse
  • Fink: fink install diffuse
BSD
FreeBSD NetBSD

Diffuse can be installed by the package management system of several BSD distributions using the following commands:

  • FreeBSD: pkg_add -r diffuse
  • NetBSD: pkg_add diffuse
