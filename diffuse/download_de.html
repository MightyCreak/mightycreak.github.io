Download

GNU/Linux
deb rpm

Die neueste Version von Diffuse wird als .deb und .rpm Paket angeboten. Diese Pakete können mit folgenden Kommandos installiert werden:

  • Debian / Ubuntu: dpkg -i diffuse*.deb
  • Fedora / OpenSUSE: rpm -i diffuse*.rpm

Alterntiv kann Diffuse über das Paketmanagement der meisten großen GNU/Linux-Distributionen installiert werden mit folgenden Kommandos:

  • Arch Linux: pacman -S diffuse
  • Debian / Ubuntu: apt-get install diffuse
  • Fedora: yum install diffuse
  • Gentoo: emerge diffuse
  • OpenSUSE: zypper install diffuse
Microsoft Windows
Win32

Das Win32 Installationsprogramm von Diffuse beinhaltet alle Abhängigkeiten die für die Benutzung auf 32-bit und 64-bit Versionen von XP / Vista / Windows 7 benötigt werden.

Mac OS X
MacPorts Fink

Pakete für Mac OS X werden von Dritten verwaltet. Diffuse kann mit folgenden Kommandos installiert werden:

  • MacPorts: port install diffuse
  • Fink: fink install diffuse
BSD
FreeBSD NetBSD

Diffuse kann über das Paketmanagement verschiedener BSD-Distributionen mit folgenden Kommandos installiert werden:

  • FreeBSD: pkg_add -r diffuse
  • NetBSD: pkg_add diffuse
